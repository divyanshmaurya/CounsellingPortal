# Futurly

**Your Future, Guided by Those Who've Been There**

Futurly is a premium mentorship marketplace and career counselling platform for students and professionals in Uzbekistan. Connect with mentors at top universities and companies worldwide.

## Tech Stack

- **Framework**: Next.js 16 (App Router) with TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Instrument Serif (headings) + DM Sans (body)

## Pages

- `/` — Homepage with hero, how it works, featured mentors, services, stats, and more
- `/mentors` — Browse and filter mentors
- `/pricing` — Pricing plans with FAQ
- `/about` — About the company, team, and mission
- `/become-mentor` — Mentor application with earnings calculator
- `/contact` — Contact form and info

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this repo to GitHub
2. Import the repo on [Vercel](https://vercel.com)
3. Vercel auto-detects Next.js and deploys — no extra config needed
